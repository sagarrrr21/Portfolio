## Project Description

This project is a responsive personal portfolio website developed using **HTML** and **CSS**.
The website showcases my skills, personal information, projects, and contact details in a structured and visually appealing format.

---

## Features

- Responsive portfolio layout
- Navigation bar with internal section links
- About Me section
- Projects showcase
- Contact form
- Hobby description section
- Social media links in footer
- Clean and modern UI design

---

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Google Fonts

## How to Run the Project

1. Download or extract the ZIP file.
2. Open the project folder.
3. Double-click **index.html**
   OR
   Right-click → Open with any web browser.
4. The portfolio website will open locally in your browser.
